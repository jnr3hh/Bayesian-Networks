Generate input data for Bayesian network construction in RIMBANET

Input: normalized, processed METSIM RNA-seq in subcutaneous adipose tissue, modules assignments from iterativeWGCNA, list of GWAS genes from haploreg nearest genes, list of GWAS genes colocalized with eQTLs, list of gene pairs from causal inference testing, list of genes with cis-eQTLs, phenotype data for each sample

Output: formatted input files for RIMBANET, inlcuding continuous data, discretized data, eQTL data and priors data.

CIT priors

Add gene -> gene pairs directly to network by using the causal inference test. If snp 1 -cisQTL-> gene A, and snp 1 -transQTL-> gene B, is the snp action on gene B mediated through gene A (snp 1 -> gene A -> gene B). You will need genotype data, cis-eQTLs fr each gene, and trans-eQTLs for each gene.

Since this takes a lot of time an memory to run, it was necessary to run as two different sets.

            trans_cutoff)
